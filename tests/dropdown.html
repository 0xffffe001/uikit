Javascript Options

Option Value Default Description
pos String 'bottom-left' Dropdown position.
mode hover | click hover Dropdown trigger behaviour.
delayShow Number 0 Delay time in hover mode before a Dropdown is shown in ms.
delayHide Number 800 Delay time in hover mode before a Dropdown is hidden in ms.
justify CSS selector false Stretch Dropdown width to a specified element.
boundary CSS selector window Referenced element to keep Dropdown's visibility.
flip false|true|'x'|'y' true Automatic Dropdown flip.
target CSS selector '.uk-dropdown' The Dropdown container selector.
cls String 'uk-dropdown' The Dropdown container class.
offset Number 0 The offset of the Dropdown container.
