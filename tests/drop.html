Javascript Options

Option Value Default Description
pos String 'bottom-left' Drop position.
mode hover | click hover Drop trigger behaviour.
delayShow Number 0 Delay time in hover mode before a Drop is shown in ms.
delayHide Number 800 Delay time in hover mode before a Drop is hidden in ms.
justify CSS selector false Stretch Drop width to a specified element.
center CSS selector false Position Drop centered to the specified element.
boundary CSS selector window Referenced element to keep Drop's visibility.
flip false|true|'x'|'y' true Automatic Drop flip.
cls String 'uk-drop' The Drop container class.
offset Number 0 The offset of the Drop container.
