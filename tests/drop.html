Javascript Options

Option Value Default Description
pos String 'bottom-left' Drop position.
mode hover | click hover Drop trigger behaviour.
delay-show Number 0 Delay time in hover mode before a Drop is shown in ms.
delay-hide Number 800 Delay time in hover mode before a Drop is hidden in ms.
hover-idle Number 100 Threshold before previously active Drop is hidden in ms.
justify CSS selector false Stretch Drop width to a specified element.
boundary CSS selector window Referenced element to keep Drop's visibilty.
prevent-flip false|true|'x'|'y' false Prevent automatic Drop flip.
target CSS selector '.uk-drop' The Drop container selector.
cls String 'uk-drop' The Drop container class.
offset Number 0 The offset of the Drop container.
