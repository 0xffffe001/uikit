Javascript Options

Option Value Default Description
source CSS selector (parentNode) The list(s) containing the items to be prioritized. By default the parent list is used as the source.
exclude CSS selector A selector to match to exclude elements from the sources.
target CSS selector (this* ul) The selector of the prioritized list. Defaults to the first UL inside this element.
container CSS selector (parentNode) The element that provides the space for the prioritized items.
container-element CSS selector Elements that consume space inside the container. Used to calculate space or detect collisions.
